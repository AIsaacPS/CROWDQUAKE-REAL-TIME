RESUMEN
================================================================================

El re-entrenamiento con el dataset ampliado (680 eventos sísmicos vs. 158
del ciclo anterior) produjo mejoras significativas en el modelo CRNN:

- Precision EQ subió de 96.49% a 98.38%: cuando el modelo dice "sismo",
  tiene razón el 98.4% de las veces. Esto significa menos falsas alarmas,
  lo cual es crítico para un sistema de alerta temprana donde cada falsa
  alarma erosiona la confianza del usuario.

- Recall EQ subió de 95.37% a 99.13%: el modelo ahora detecta el 99.1%
  de los sismos reales, fallando solo en el 0.87% de los eventos. Para
  alerta temprana, no perder sismos es la prioridad número uno.

- AUROC de 0.9998: prácticamente perfecto. Este valor es comparable con
  los mejores resultados del paper original de CrowdQuake (KDD 2020),
  que usó 8,973 eventos. Con solo 680 eventos ya alcanzamos rendimiento
  de nivel publicación.

- El F1 de 98.75% confirma un excelente balance entre precision y recall,
  lo que indica que el modelo está listo para pruebas en tiempo real.

El ANN baseline empeoró con más datos (AUROC 0.9613 → 0.9476, Accuracy
91.43% → 74.39%), confirmando que las 3 features manuales (IQR, ZC, CAV)
no capturan la complejidad de los patrones sísmicos tan bien como la CRNN,
que aprende directamente de la forma de onda cruda. Esto valida la decisión
de usar la CRNN como modelo principal para el sistema de detección.
